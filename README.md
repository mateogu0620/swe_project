# TicketScraper

To build production, type `make prod`.

To create the env for a new developer, run `make dev_env`.

A flask rest API server that searches for nearby converts(from Ticketmaster, EventBrite, SeatGeek, etc) and stores and presents them to users. We also allow users to filter events based on factors such as userâ€™s location, preferred genre, ticket prices , etc..

The goal is to make a website that helps users find new and exciting events near them to explore in a single place.


## Requirements

List events/concerts based on a set of filters

Potential Filters:
    Location
    Date
    Price
    Venue
    Artist
    Genre
    Ticket Availability

Allow viewing and sorting events by a given filter

Link event/concerts to their according site where we got the information from
